# OSPeCon / UOCRA - Proyectos Base

Este repositorio unificado contiene las plantillas base estandarizadas y actualizadas (Año 2026) para los proyectos de la organización.

## 📦 Estructura
- `/Frontend` : Contiene el Frontend Base. Migrado a **Vite/Webpack 5**, **Lit 3** y **Redux Toolkit (RTK)**. Incluye componentes nativos de interfaz de usuario basados en **Material Design 3** con soporte para *Dark Mode*.
- `/Backend` : Contiene el Backend Base. Arquitectura Limpia basada en **MediatR**, CQRS y actualizado completamente a **.NET 8.0 / .NET 10.0 LTS**. 

Ambos proyectos se encuentran listos para ser clonados e iniciados de manera independiente o conectados entre sí mediante Proxy Inverso para evadir problemas de CORS en desarrollo.
